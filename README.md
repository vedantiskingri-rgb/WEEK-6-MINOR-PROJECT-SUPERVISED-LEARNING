# WEEK-6-MINOR-PROJECT-SUPERVISED-LEARNING
This project aims to predict whether a movie will be a success (a “Hit”) using historical data such as rating, genre, country, duration, budget, and other features. We first perform exploratory data analysis, encode categorical variables, and create a binary target Hit (e.g., high‑rated movies = 1, others = 0). Then we train a Logistic Regression model on the processed features, evaluate it using accuracy, precision, recall, F1 score, and a confusion matrix, and interpret which factors most influence movie success. The final goal is to build an interpretable model that can help stakeholders make data‑driven decisions about production, marketing, and release strategy
